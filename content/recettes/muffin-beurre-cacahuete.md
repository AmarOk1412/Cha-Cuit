# Ingrédients

+ 220g de beurre de cacahuète croustillant
+ 2 œufs
+ 2 yaourt nature
+ 120g de pépites de chocolat
+ 220g de farine
+ 100g de sirop d'érable
+ 1 c à soupe levure instantanée
+ 50g de beurre


# Équipement

+ 16 moule à muffins
+ 1 Four
+ 1 Bol

||

# Préparation

1. Dans un bol, mélanger la farine, la levure délayée, le sirop d'érable, les yaourts, le beurre de cacahuète.
2. Ajoutez les œufs
![La pâte homogène](/img/muffin-beurre-cacahuete/muffin-beurre-cacahuete-step-2.jpg)
3. Ajoutez les pépites de chocolat
4. Beurre les moules et les remplir de pâte à muffins jusqu'au 3/4.
5. Cuire 25min à 180°C.
